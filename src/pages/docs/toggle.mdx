# Toggle Component

Whether it is used for a light / dark theme or turning a setting on / off, toggles have become staples in modern UI. 

## Example usage

```jsx
<Switch defaultIsChecked />
```

## Colors

```jsx
<Stack isInline>
  <Switch color="primary" />
  <Switch color="blue" />
  <Switch color="red" />
  <Switch color="yellow" />
  <Switch color="green" />
</Stack>
```

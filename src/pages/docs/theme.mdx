## Fonts, Sizes, Weights

### Font Family

For the body text and headings, I've opted for one font - [Inter UI](https://rsms.me/inter/). Its a fantastic modern font made by Figma.

For code, I'm using the `Fira Code`, a google font.

### Sizes

Font sizes are currently named for ease of use. This could be changed in the future as names are hard to add along the way like this.

```jsx
<Stack>
  <Text fontSize="xs">Text Placeholder</Text>
  <Text fontSize="sm">Text Placeholder</Text>
  <Text fontSize="md">Text Placeholder</Text>
  <Text fontSize="lg">Text Placeholder</Text>
  <Text fontSize="xl">Text Placeholder</Text>
  <Text fontSize="2xl">Text Placeholder</Text>
  <Text fontSize="3xl">Text Placeholder</Text>
  <Text fontSize="4xl">Text Placeholder</Text>
  <Text fontSize="5xl">Text Placeholder</Text>
</Stack>
```

### Weights

For simlicity's sake, I decided to retain the 100-900 scale for text weights as well.

```jsx
<Stack>
  <Text fontWeight={100}>Text Placeholder</Text>
  <Text fontWeight={200}>Text Placeholder</Text>
  <Text fontWeight={300}>Text Placeholder</Text>
  <Text fontWeight={400}>Text Placeholder</Text>
  <Text fontWeight={500}>Text Placeholder</Text>

```

## Space

I based my scale off of [Gitlab's Pajamas documentation](https://design.gitlab.com/layout/spacing) which uses geometric progression. They have a 12px outlier, however I didn't feel it was needed.

```json
space: {
  "0": "0",
  "1": "0.125rem", // 2px
  "2": "0.25rem", // 4px
  "3": "0.5rem", // 8px
  "4": "1rem", // 16px
  "5": "1.5rem", // 24px
  "6": "2rem", // 32px
  "8": "2.5em", // 40px
  "10": "3rem", //48px
  "11": "3.5rem", // 56px
  "12": "4rem", // 64px
  "16": "6rem", // 96px
  "20": "8rem", // 128px
  "24": "11rem", // 176px
  "32": "16rem", // 256px
}
```
